1. Basic Media Query

  • Change styles based on screen width.
  • max-width → applies when screen is smaller than given width.
Code:

.demo-box {
  background: teal;
}

/* Mobile (≤600px) */
@media (max-width: 600px) {
  .demo-box {
    background: red;
    width: 100%;
  }
}
Output Preview:

➡ Default: teal | ≤600px: red + full width

2. Orientation

  • Detect if device is in portrait or landscape mode.
Code:

.orientation-box {
  background: yellow;
}

/* Landscape mode */
@media (orientation: landscape) {
  .orientation-box {
    background: blue;
  }
}
Output Preview:
Rotate Screen

➡ Yellow (default) | Blue if in landscape

3. Multiple Conditions

  • Combine conditions with and.
  • Example: Apply styles only for tablets (600px–900px).
Code:

.multi-box {
  background: #ff9a9e;
}

/* Tablets between 600px and 900px */
@media (min-width: 600px) and (max-width: 900px) {
  .multi-box {
    background: purple;
  }
}
Output Preview:
Resize

➡ Pink default | Purple if width 600–900px
