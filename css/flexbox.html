Flex Item Properties (Child)

  • order: changes the sequence of boxes.
  • flex-grow: controls which box expands more.
  • flex-shrink: controls which box shrinks more.
  • flex-basis: starting size of the box.
  • flex: shorthand for grow, shrink, basis.
  • align-self: one box can ignore the group‚Äôs alignment.
Code:
.item1 { order: 2; flex-grow: 1; flex-basis: 80px; } .item2 { order: 1; flex-grow: 2; flex-basis: 60px; } .item3 { order: 3; flex-grow: 1; flex-basis: 50px; }

- order: Even if box 2 is written later, it can appear first.
- flex-grow: Bigger number = box grows more when extra space is available.
- flex-shrink: Bigger number = box shrinks more if space is tight.
- flex-basis: Starting width/height of a box.
- align-self: One box can choose its own position (top, bottom, middle).

Imagine three kids sharing a pizza üçï ‚Äî who gets the bigger slice depends on flex-grow.
