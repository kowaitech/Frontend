Flex Container: display

  • Sets element as flex container: flex or inline-flex.
Code:
<div style="display: flex;"> <div>Item 1</div> <div>Item 2</div> <div>Item 3</div> </div>
Output Preview:
Item 1
Item 2
Item 3
  • Inline-flex behaves like flex but inline.
Code:
<div style="display: inline-flex;"> <div>Item A</div> <div>Item B</div> </div>
Output Preview:
Item A
Item B

Flex Container: align-items

  • stretch (default): items stretch to fill container cross-axis
Code:

Item 1
Item 2
Item 3
