1. px (Pixels)

  • Fixed size unit.
  • Does not change with screen or text size.
Code:
<div style="font-size:16px;">Text with 16px font size</div>
Output Preview:
Text with 16px

2. % (Percentage)

  • Relative to the parent element.
  • Useful for responsive layouts.
Code:
<div style="width:50%; background:lightblue;">Width 50% of parent</div>
Output Preview:
Parent Box
50% Width

3. em (Relative to parent font size)

  • 1em = 100% of parentâ€™s font size.
  • Grows/shrinks based on parent.
Code:
.parent { font-size: 16px; } .child { font-size: 2em; } <!-- = 32px -->
Output Preview:
Parent (16px)
Child 2em

4. rem (Relative to root font size)

  • 1rem = based on <html> font-size.
  • More consistent than em.
Code:
html { font-size: 16px; } div { font-size: 1.5rem; } <!-- = 24px -->
Output Preview:
Text 1.5rem

5. vh (Viewport Height)

  • 1vh = 1% of the viewport height.
  • 100vh = full screen height.
Code:
<div style="height:20vh; background:orange;">20vh height</div>
Output Preview:
20vh Tall

6. vw (Viewport Width)

  • 1vw = 1% of the viewport width.
  • 100vw = full screen width.
Code:
<div style="width:50vw; background:green;">50vw wide</div>
Output Preview:
50vw Wide
