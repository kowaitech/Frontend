1. Font Family

  • Defines the typeface of the text.
  • Best practice: provide fallback fonts like Arial, sans-serif.
Code:
p { font-family: Arial, sans-serif; }
Output Preview:

This is Arial font.

2. Font Size

  • Controls the size of the text.
  • Values: px, em, rem, etc.
Code:
p { font-size: 30px; }
Output Preview:

This is 30px text.

8. Font Units (px, em, rem)

  • px → Absolute pixels (fixed size, doesn’t scale).
  • em → Relative to parent’s font size (1em = 100%).
  • rem → Relative to root html font size (more consistent).
Code:
p.px-text { font-size: 20px; } p.em-text { font-size: 2em; /* 2 × parent size */ } p.rem-text { font-size: 1.5rem; /* 1.5 × root html size */ }
Output Preview:

This is 20px text (fixed)

This is 2em text (relative to parent)

This is 1.5rem text (relative to root)

3. Font Weight

  • Defines the thickness of characters.
  • Common values: normal, bold, numbers (100–900).
Code:
p { font-weight: bold; }
Output Preview:

This is bold text.

4. Text Align

  • Controls horizontal alignment of text.
  • Values: left, center, right, justify.
Code:
p { text-align: center; }
Output Preview:

This text is centered.

5. Text Decoration

  • Adds or removes decoration lines on text.
  • Values: none, underline, overline, line-through.
Code:
a { text-decoration: none; }
Output Preview:
This link has no underline

6. Text Transform

  • Controls capitalization of text.
  • Values: uppercase, lowercase, capitalize.
Code:
p { text-transform: uppercase; }
Output Preview:

This text is uppercase

7. Word Spacing

  • Defines space between words.
  • Value in px, em, etc.
Code:
p { word-spacing: 5px; }
Output Preview:

This text has extra spacing
