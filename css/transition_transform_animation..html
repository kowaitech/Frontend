Transform

Transform: A CSS property used to instantly change the shape, size, position, or orientation of an element. It does not animate by itself, the effect happens immediately. Common transform functions include:
  • rotate(angle) → Rotates the element (e.g., rotate(45deg)).
  • scale(x, y) → Resizes the element (e.g., scale(1.5) makes it 150%).
  • translate(x, y) → Moves the element (e.g., translate(50px, 20px)).
  • skew(x-angle, y-angle) → Tilts the element (e.g., skew(20deg, 10deg)).

1. Rotate

  • Rotates an element clockwise or counterclockwise.
  • Angle is given in deg or turn.
  • Real life: Rotating a photo frame.
Code:
<div style="transform: rotate(45deg);"> Rotated Box </div>

2. Scale

  • Resizes an element (bigger or smaller).
  • scale(1.5) makes it 1.5 times larger.
  • Real life: Zooming into an image.
Code:
<div style="transform: scale(1.5);"> Scaled Box </div>

3. Translate

  • Moves the element from its original position.
  • translate(x, y) → x = horizontal, y = vertical.
  • Real life: Shifting a box on a table.
Code:
<div style="transform: translate(50px, 30px);"> Moved Box </div>

4. Skew

  • Tilts (slants) an element along X or Y axis.
  • skew(20deg, 10deg) → tilt horizontally and vertically.
  • Real life: Looking at a rectangle from an angle.
Code:
<div style="transform: skew(20deg, 10deg);"> Skewed Box </div>

Transition

  • Makes CSS changes happen smoothly over time.
  • Needs a trigger like :hover or JS event.
  • Real life: A curtain opening slowly instead of instantly.
Code:
<style> .transition-box { background: skyblue; transition: transform 0.5s, background 0.5s; } .transition-box:hover { transform: scale(1.3); background: dodgerblue; } </style> <div class="transition-box"> Hover Me </div>
Output Preview:
Hover

Animation

  • Creates continuous or multi-step movement.
  • Runs automatically without hover.
  • Uses @keyframes to define motion.
  • Real life: A bouncing ball.
Code:
<style> @keyframes bounce { from { transform: translateY(0); } to { transform: translateY(50px); } } .animation-box { background: seagreen; animation: bounce 1s infinite alternate; } </style> <div class="animation-box"> Bounce </div>
Output Preview:
Bounce

Transform vs Transition vs Animation

  • Transform: Instant effect (e.g., rotate, scale, move).
  • Transition: Smooth effect when triggered (e.g., hover).
  • Animation: Continuous/automatic effect using @keyframes.
