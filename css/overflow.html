1. Overflow: Visible

  • Default value.
  • Content overflows and is not clipped.
Code:
<div style="overflow: visible;"> Long text goes beyond the box... </div>
Output Preview:
This is a very long text that will overflow outside the box without any restriction.

2. Overflow: Hidden

  • Extra content is cut off.
  • No scrollbars are shown.
Code:
<div style="overflow: hidden;"> Extra text is hidden </div>
Output Preview:
This is a very long text that gets hidden beyond the box. You cannot scroll to see the rest.

3. Overflow: Scroll

  • Scrollbars are always visible, even if not needed.
Code:
<div style="overflow: scroll;"> Always shows scrollbars </div>
Output Preview:
This is a very long text that will make the box scrollable. Scrollbars are always shown, even if content fits inside.

4. Overflow: Auto

  • Scrollbars appear only when needed.
Code:
<div style="overflow: auto;"> Scrollbars appear only if content is too large </div>
Output Preview:
This is a very long text that will only show scrollbars if the content doesn't fit inside the box.
